# Alien-Hotdog-Stand-VideoGame

This is an indie videogame passion project that is a mix between papas pizzeria and an alien themed puzzle game, where you have to use context clues and a trusty handbook to figure out what each customer wants to eat!

This current stable build of the game features a completed topping system, with a working food preference calculation system underneath. Development of the individual puzzles will be uploaded once the art is completed (which I all do myself!). An example video of this current path can be found [here](https://www.youtube.com/watch?v=JxRUgSLXZN8)
